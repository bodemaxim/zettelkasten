# Задание на рефакторинг

Мы не объявляем типы во .vue файле. Если внутри *.vue объявлен тип (при помощи ключевого слова `type`), его надо перенести в *.types.ts в той же папке. Если в *.vue есть объявленние типов, а файл *.types.ts отсутствует - создай этот файл и перенеси в него тип, используя `export type`.

Например, есть файл `edit-device-panel.vue` и в нем объявлен тип. Тогда надо перенести объявление типа в `edit-device-panel.types.ts` в той же папке.

Если объявленный тип - это не тип пропс, эмитов или модели и в файле существует регион `#region API компонента`, объявляй этот новый тип ниже строки `#endregion API компонента`.

Не вноси никаких изменений кроме указанных в этом файле.
