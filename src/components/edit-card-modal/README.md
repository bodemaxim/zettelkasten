# Загрузка квизов через JSON (вставка в textarea)

Для квиза используются поля базовой карточки (`CardEditable`) и поля из таблицы `quizzes`.

Ключевые для квиза поля:
- `task` — задание
- `prefilled_answer` — текст, который будет заранее подставлен пользователю в редактор ответа
- `study_points` — очки приоритета/сложности

Пример JSON для задания: `Напиши функцию в python, складывающую два числа`

```json
{
  "title": "Сложение двух чисел (Python)",
  "createdAt": null,
  "type": "quiz",
  "folders": "[]",
  "links": [],
  "text": "def add(a, b):\n    return a + b",
  "study_points": 0,
  "task": "Напиши функцию в python, складывающую два числа",
  "prefilled_answer": "def add(a, b):\n    "
}
```

Примечания:
- `createdAt: null` — дату проставляет БД при сохранении; в JSON всегда `null`.
- `folders` в текущей модели — строка, содержащая JSON-массив UUID папок (поэтому `"[]"`).
- `text` — “правильный ответ” (показывается после отправки решения).
